ACME Corp

Build the homepage for Acme Anvil Corporation using HTML and CSS. Please do so without using classes or IDs.

  • Hi-res screenshot
  • The logo image (jpg)
  • The background image (png)
  • To center a container (e.g. <main>) set margin-left and margin-right to auto and set a width. You can set the width to a percentage like 60% or so. Try also playing with max-width.
  • Font-family is "Lucida Grande".
  • Use the "Digital Color Meter" under Utilities on your Mac to figure match the colors. When you have your mouse over the color you want, use Command + Shift + C to copy the color to your clipboard.
  • The buttons and links don't have to work.
  • Take advantage the <header>, <main>, and <footer> elements to organize your content.
  • Use a <button> element for the "Sign up" button.
